¿Qué es y que función cumple el CSS?

CSS son las siglas para Cascading Style Sheets. es un lenguaje que maneja el diseño y presentacion de las paginas web.

¿Cómo se incorpora el CSS dentro del HTML?

Las reglas de estilo se pueden agregar directamente a cualquier elemento HTML. Para hacer esto, simplemente hay que agregar un atributo de estilo al elemento y luego indicar las reglas deseadas en la misma línea. Este método presenta las siguientes ventajas: Los estilos en línea anulan los estilos externos.

¿Cuál es la anatomía de una regla CSS?

Esta selecciona el o los elementos a dar estilo. Para dar estilo a un elemento diferente, solo cambia el selector.

¿Cuáles son los tipos de selectores que puede utilizar CSS?

Existen muchos tipos diferentes de selectores, algunos de los que puede utilizar son: selector de elemento, selector de identificacion, selector de clase, selector de atributo y selector de pseudoclase.

es una interfaz de programación para los documentos HTML y XML. Facilita una representación estructurada del documento y define de qué manera los programas pueden acceder, al fin de modificar, tanto su estructura, estilo y contenido.

¿En qué puede ayudarnos javascript en una página web?

Javascript nos puede ayudar a que nuestra pagina web sea mas interactiva con la persona que la utilize, con este se puede agregar links, cambiarle la fuente a las palabras, subrayar un titulo, herramientas que nos ayudan a hacerla mas llamativa a la pagina web.
